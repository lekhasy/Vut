02 — Core rules
R-01
Caps discipline
All display text (H1–H3), labels, eyebrows, buttons, and nav must be ALL-CAPS via text-transform: uppercase. Body and subhead text uses sentence case only. Never title case.
R-02
Tracking inversion
Large headlines use minimal tracking (0.02–0.04em). Small labels use heavy tracking (0.12–0.18em). This is counterintuitive but produces the precision-engineering look.
R-03
Line-height compression
Headlines run below 1.0 line-height (0.92–0.95). This creates stamped, pressurized text blocks. Never apply loose leading to display text.
R-04
Weight restraint
Use only three weights: 400 for body, 600 for labels and CTAs, 700 for display headlines. Weight 300 and 500 are unused. Never mix 600 and 700 at the same hierarchy level.
R-05
Color hierarchy
Headlines: #F0F0EE (near-white). Body / subhead: #888884 (muted gray). Labels / eyebrows: #444442 (dim). One primary color only — no accent palette. Reserve color for status signals.
R-06
No decorative styling
No italic, no underline on body text, no bold mid-sentence. Emphasis is achieved through size and uppercase only. The font does the work — no additional decoration needed.

}
